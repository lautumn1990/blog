每个人都喜欢在 Windows 中快速完成任务。如果鼠标往往会减慢您的速度，那么这里有一份完整的键盘命令列表，您可以使用它来完成通常使用鼠标执行的几乎所有任务。

<!--more-->

## 常用快捷键

| --------------------------------- | ------------------------------------- |
| Power menu                        | Windows + X 或者 右击开始菜单         |
